--- layout: default title: GitHub Pages nav_order: 3 has_children: true --- # GitHub Pages GitHub pages is free, static, websites hosted by [GitHub](https://help.github.com/en/github/working-with-github-pages). This means you can do fun things in publishing, but the final site needs to be static content with client side javascript. ## Tech Stack Backend / Publishing - Jekyll - Ruby - Markdown Frontend - Static HTML - Javacript ## Hosting The default site is https://username.github.io/reponame but you can also setup your own domain. For more information about [configuring a custom domain for your GitHub Pages site, click here](https://help.github.com/en/github/working-with-github-pages/configuring-a-custom-domain-for-your-github-pages-site)
